# jackdan_react_echarts
 
------

## 功能列表

- [X] 自定义画布
- [X] 可拖拽效果
- [X] 自适应屏幕分辨率(目前支持是1080)
- [X] 三列布局
- [X] 中国地图
- [X] 北京市

```

 npm install
 # recommended install dependencies
 cnpm install
 # serve with hot reload at http://localhost:12347/#/main/tworesponsibilities/index
 yarn run dev
 # rebuild it
 yarn run build
```
